GitHub Copilot es una herramienta de inteligencia artificial, actualmente en revisión técnica, que puede hacer recomendaciones a los programadores cuando escriben código. GitHub Copilot funciona con OpenAI Codex, un nuevo sistema de inteligencia artificial creado por OpenAI. Microsoft ya lo ha implementado como una extensión en Visual Studio Code y lo integrará en el producto VS comercial. Por ahora, solo algunos desarrolladores pueden acceder a él.

Puede completar líneas de código o escribir funciones completas, transformar comentarios descriptivos en código, autocompletar código repetitivo o crear pruebas unitarias para sus métodos. Copilot se basa en un modelo de lenguaje similar al GPT-3, por lo que se ha adaptado al código en el que se ha entrenado. Pero, a medida que lo usa, comienza a "comprender" su estilo y se adapta a usted. Y si no le gusta la primera sugerencia, puede pedirle que proporcione soluciones top-k. Funciona mejor con Python, JavaScript, TypeScript, Ruby y Go, pero "comprende docenas de lenguajes". Sin embargo, debido a que a veces falla, puede ser más útil al explorar nuevas bibliotecas o escribir en un lenguaje de programación desconocido.

Los desarrolladores de GitHub realizaron un experimento para medir la precisión de la herramienta. Se probó en un conjunto de funciones de Python que tienen una buena convergencia de prueba en repositorios de código abierto. Se eliminaron todos los cuerpos de las funciones y el único contexto proporcionado fueron los nombres de las funciones y las cadenas de documentos. Copilot pudo completarlos correctamente el 43% del tiempo, y la precisión aumentó al 57% después de 10 intentos. Esta es realmente una hazaña notable, ya que la herramienta genera código de trabajo que puede utilizar en sus proyectos. Dado que millones lo usarán una vez que se haga público, acelerará significativamente el desarrollo.
